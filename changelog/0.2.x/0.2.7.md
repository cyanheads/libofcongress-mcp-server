# 0.2.7 — 2026-06-02

## Changed

- **`@cyanheads/mcp-ts-core`** `^0.9.16 → ^0.9.21` — HTTP transport now carries fresh request + trace/span IDs per request instead of the frozen boot context; `fetchWithTimeout` strips query-string secrets (e.g. `?api_key=`) from error messages and logs; `withRetry` fails fast on non-retryable errors and `ctx.fail` auto-populates the `retryable` flag.
- **README client-config keys** renamed from bare `libofcongress` to the full package name `libofcongress-mcp-server`; `bunx`/`npx` args updated to the scoped package name `@cyanheads/libofcongress-mcp-server@latest`.
- **Skills** re-synced with framework `0.9.21`: `add-tool`, `add-service`, `api-canvas`, `api-context`, `api-linter`, `api-utils`, `design-mcp-server`, `release-and-publish` updated; `api-mirror` and `orchestrations` skills added.
- **`scripts/devcheck.ts`** updated; `scripts/check-skill-versions.ts` and `scripts/release-github.ts` added; `release:github` npm script added.

## Dependencies

- `@cyanheads/mcp-ts-core` ^0.9.16 → ^0.9.21
- `vitest` ^4.1.7 → ^4.1.8
